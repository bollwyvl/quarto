Quarto is a frontend for Jupyter with the goal of beautiful computing.

beautiful

Quarto is inspired by great literary artifacts.

  • Galileo's Sidereal Messenger
  • Jodorowsky's Dune

computing

All of this is being stored in an Jupyter Notebook, which means you can use it with any Jupyter Kernel, easily share with with colleagues, or the world.

At any point, you can create an indented or fenced code block, and get a text editor.

Additionally, you can perform notebook meta commands (like save, search/replace) with the console: try opening it by pressing ctrl+esc.
